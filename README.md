# emoji-grass
Displays the grass of GitHub by emoji
## install
```
$ npm install -g emoji-grass
```
## Usage
```
Usage: emoji-grass [options] [command]

Options:
  -h, --help                    display help for command

Commands:
  display [options] <username>  Display the grass of GitHub by emoji
  settoken <token>              Set up the personal access token of GitHub
                                Generate a token in https://github.com/settings/tokens
  help [command]                display help for command

```
